---

### 要下载一个网站的CA（Certificate Authority，证书颁发机构）证书，通常有两种常见的方式：通过浏览器直接下载和通过命令行工具下载。下面是这两种方法的具体步骤：

### 下载证书

##### 1. 通过浏览器直接下载：

1. 打开网站：
   在浏览器中打开你想要获取证书的网站，确保使用的是 HTTPS 协议。
2. 查看证书信息：
   在浏览器地址栏中，点击锁形图标（通常位于网址左侧）。
   选择“证书”或“证书信息”选项（或“连接是安全的” -> “证书”）。
3. 切换到详细信息标签页，选择根CA证书（最顶层）
4. 导出证书：
   浏览器将显示证书的详细信息。
   根据不同的浏览器，选项可能略有不同。通常你可以选择“查看证书”或“详细信息”，然后找到“导出”或“另存为”选项来保存证书。

##### 2. 通过命令行工具下载：

1. 安装openssl：
   确保你的系统中已安装了 OpenSSL。如果尚未安装，可以在官方网站上找到安装指南：[OpenSSL](https://openssl-library.org/) 官网
2. 下载证书：
   打开终端或命令提示符。
   使用以下命令下载证书：

   ~~~shell
      openssl s_client -connect example.com:443 -servername example.com < /dev/null \
      2>/dev/null | openssl x509 -outform PEM -out example.com.pem
   ~~~

   这里的 example.com 需要替换成你想要获取证书的网站域名。
3. 查看导出的证书
   ~~~shell
   openssl x509 -in your_cert.crt -text -noout
   ~~~
4. 在你的证书内容中这一段
   ~~~
   Authority Information Access:
       CA Issuers - URI:http://secure.globalsign.com/cacert/gsrsaovsslca2018.crt
       OCSP - URI:http://ocsp.globalsign.com/gsrsaovsslca2018
   ~~~
5. 用浏览器打开
   ~~~
   http://secure.globalsign.com/cacert/gsrsaovsslca2018.crt
   ~~~

### 导入证书

1. 导入网站证书
   ~~~shell
   keytool -import -alias server-cert -file server.crt -keystore $JAVA_HOME/jre/lib/security/cacerts -noprompt
   ~~~
2. 导入中间CA证书
   ~~~shell
   keytool -import -alias intermediate-ca -file intermediate.crt -keystore $JAVA_HOME/jre/lib/security/cacerts -noprompt
   ~~~
3. 导入根CA证书
   ~~~shell
   keytool -import -alias root-ca -file root.crt -keystore $JAVA_HOME/jre/lib/security/cacerts -noprompt
   ~~~
4. 证书链是
   ~~~text
   网站证书 ← GlobalSign RSA OV SSL CA 2018 ← GlobalSign Root CA
   ~~~
   只需要导入中间 CA + 根 CA 两个证书就够了！网站证书可导可不导。
5. 如果证书是放到SecretManage中，可以讲证书链的pem合并到一个文件中，从上到下是：
   ~~~text
   网站证书 ← GlobalSign RSA OV SSL CA 2018 ← GlobalSign Root CA
   ~~~
   如下：
   ~~~text
   -----BEGIN CERTIFICATE-----
    网站证书
   -----END CERTIFICATE-----
   -----BEGIN CERTIFICATE-----
    中间CA证书
   -----END CERTIFICATE-----
   -----BEGIN CERTIFICATE-----
    根证书
   -----END CERTIFICATE-----
   ~~~

### 补充

crt文件转pem

~~~shell
openssl x509 -in gsrsaovsslca2018.crt -out gsrsaovsslca2018.pem -outform PEM
~~~
